SKELETON INC. / INFORMATION SECURITY POLICY

情報セキュリティ基本方針

制定: 2026年7月10日 / 最終更新: 2026年7月10日

スケルトン株式会社 (以下、 当社) は、 お客様からお預かりした情報資産、 及び当社の情報資産を、 事故・災害・犯罪等の脅威から守り、 お客様ならびに社会の信頼に応えるべく、 以下の方針に基づき全社で情報セキュリティに取り組みます。

1. 経営者の責任

当社は、 経営者主導で組織的かつ継続的に情報セキュリティの改善・向上に努めます。

2. 社内体制の整備

当社は、 情報セキュリティの維持及び改善のために責任者を定め、 情報セキュリティ対策を社内の正式な規則として定めます。

3. 従業員の取組み

当社の従業員は、 情報セキュリティのために必要とされる知識、 技術を習得し、 情報セキュリティへの取り組みを確かなものにします。

4. 法令及び契約上の要求事項の遵守

当社は、 情報セキュリティに関わる法令、 規制、 規範、 契約上の義務を遵守するとともに、 お客様の期待に応えます。

5. 違反及び事故への対応

当社は、 情報セキュリティに関わる法令違反、 契約違反及び事故が発生した場合には適切に対処し、 再発防止に努めます。

6. 具体的な取り組み

技術的対策

  • お客様データを保管するクラウドインフラ (Google Cloud / Firebase, asia-northeast1) の 暗号化通信 (TLS 1.2以上) を全通信で強制
  • アクセス権限は role-based で最小権限の原則を適用
  • 認証情報 (API キー、 パスワード等) は Google Secret Manager 上でのみ管理し、 コードベースに含めない
  • 本番環境への変更は 段階的リリースと 事後検証を必須化

組織的対策

  • 全社員に対する 情報セキュリティ 基本方針・関連規程 の 周知徹底
  • 個人情報を含む情報資産の 取扱い ルール の 明文化
  • 取引先・業務委託先との 秘密保持契約 の 締結
  • 事故発生時の 対応フロー を あらかじめ定め、 迅速な対応を可能にする体制の整備

物理的対策

  • 執務スペース の 施錠管理
  • 紙媒体の 重要情報 の 施錠保管、 廃棄時の シュレッダー処理
  • 離席時の 画面ロック の 徹底

継続的改善

  • 定期的な 自己点検 (IPA「5分でできる情報セキュリティ自社診断」 準拠) の 実施
  • 脅威動向 (IPA公表情報等) の 継続的な把握
  • 新たな脅威 に対する 対策 の 適時追加

7. 情報セキュリティ認証

当社は、 IPA (独立行政法人情報処理推進機構) の 「SECURITY ACTION」 制度に 準拠した 情報セキュリティ対策 の 自己宣言を 行っています。 本基本方針の 公開 は、 その 一環です。
